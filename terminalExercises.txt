How do you jump between words in the terminal?
   - Option/Alt + left/right 

How do you get to the end of a line in terminal?
  - Command/Control + e

How do you move your cursor to the beginning in terminal?
  - Command/Control + a

How do you delete a word (without pressing backspace multiple times) in terminal?
  - Command/Control + w

What is the difference between a terminal and shell?
  - Terminal is the application that contains the CLI. Shell is the program that proccesses commands and returns output.

What is an absolute path?
 - Absolute paths start with the root directory and shows the whole path to a file/directory. 

What is an relative path?
 - Relative paths show you the path relative to which directory you are currently in.

What is a flag? Give three examples of flags you have used.
 - Flags give us diffent options to use with commands for more involved commands. examples: -m, -u, -a

What do the r and f flags do with the rm command?
 - -f attempts to remove a file without needing permissions. -r attempts to remove a file hierachy rooted in each file argument.
